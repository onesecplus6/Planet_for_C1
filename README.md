# Planet_for_C1
拥有全景星空与真实光照的星球桌面

载入天空盒全景图和星球贴图，在屏幕上生成一个星球。
第一次打开时会进行校准，只用一只眼睛正对小屏幕，鼠标移动使左右两侧的绿条与中间的白色处于同一水平线，点击鼠标完成校准。后续想重新校准可以按P键。如果是黑屏，请先启动OpenstageAI。
鼠标左键拖动旋转星球，右键拖动旋转背景。鼠标滚轮可以放大缩小星球，WS控制入屏/出屏效果，AD控制自旋速度，EQ顺时针或逆时针旋转星球，R还原。上下方向键切换星球，左右方向键切换背景。ESC或在底部任务栏退出。
想要增加背景，可以进入根目录的skybox文件夹，在里面添加自己的天空盒全景图。想要增加星球，可以进入根目录的planet文件夹，在里面添加自己的星球贴图。
